Hello, RIT!

All RIT IDs carry a unique university ID. We can safely take advantage of this technology to use it for authentication. Think about letting your club members check-in with just a swipe; or building a cool system with this RIT ID Auth API.


Security is important

The most important thing is to be safe. RIT Student ID numbers are stored in the system only and only hashed with SHA-512 - virtually impossible to decode in asymptotic linear time.
